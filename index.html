Jot

A Discord Moderation Bot

Features

Add to server

A bot with a primary focus on moderation tools: Issue warnings, mutes, kicks and bans with an auto-mod, logging and more.

Developed and tested alongside moderating a server with 30,000+ members.

Excellent logging

Built-in DM relays for user appeals, hyperlinked in the logs. Auto-mod quotes messages so you don't have to go look. Notifications when old users you've dealt with rejoin the server.

Easy to configure

No intricate web dashboard, use !setup and !automod and run through simple guides that are built-in to Discord.

Smart permissions

Designed to use the bare minimum permissions to function. Want tiered moderator roles with lesser permissions? This bot does it automatically.
