Donate

Hey there! If you like the bot and want to show support with a small donation, you can!


Or, if you're interested in upgraded features, those exist too! Read on below.

What does upgrading do?

Supporting the bot on Patreon grants you upgrade tokens, which you apply to Discord servers with the /upgrade command. An upgraded server gets the following features:
  • No command cooldowns
  • Allows full customization of button roles
  • Enables the server link feature
    * Only a single server needs to be upgraded, you can link 4 servers together with 1 upgrade token

Additionally, any supporters (not just Patreon) get a fancy role and a private category in the Jot Discord server!

Where do I sign up?

At this time, only Patreon members will receive upgrade tokens. You can become a patron with the button below!
If your Discord account is linked in your Patreon account settings, the bot will recognize you within 5 minutes.

Become a Patron!

Don't want to link your account? Have any issues? DM an administrator in the Jot Discord server to get set up!
Here is how the system works:
  • Become a patron, link your Discord account in your Patreon account settings and run /upgrade in one of your servers.
  • Select one of your tokens in that dropdown box to apply it to your server. You can freely move tokens between servers as many times and as often as you want!
  • Whenever Patreon charges you, the tokens you are granted will last (or be extended by) 32 days.

What happens if I stop being a patron?

When your tokens expire (detailed in the panel above):
  • Command cooldowns come back
  • Existing button roles will remain fully functional, you just can't fully customize any new ones you create
  • Servers will stay linked in the configuration, the links will simply not function until one of the linked servers becomes upgraded again
